Getting Started with Rivvix

Learn how to set up and start using Rivvix, the ultimate Discord bot, with these easy steps.

Setup Instructions

Step 1: Invite Rivvix to Your Server

Click here to invite Rivvix to your server. Make sure to grant the necessary permissions for Rivvix to function properly.

Step 2: Configure Settings

Once Rivvix is in your server, use the /setup command to configure the basic settings. Follow the prompts to set up features like logging, moderation, and more.

Step 3: Explore Commands

Use the /help command to view a list of available commands and their descriptions. Check out the features section to see what Rivvix can do!

FAQ

How do I use the commands?

Type /help to get a list of all available commands and their usage.

What permissions does Rivvix need?

Rivvix needs permissions to manage messages, read message history, and more. Ensure the bot has the appropriate permissions for full functionality.

Can I customize Rivvix?

Yes, Rivvix offers various customization options through the /setup command.
